# 🌍 에너지 보드게임 아레나

세 가지 게임을 한 사이트에서 즐기는 **웹 전용 실시간 멀티플레이** 보드게임 모음입니다.

| 게임 | 인원 | 설명 |
|------|------|------|
| 🌐 **가이아** | 2–4인 | 17×17 맵에서 자원을 차지하는 하이브리드 전략전. 화석연료 vs 재생에너지, 외교·동맹, 초대형 인프라, 돌발 기후 이벤트. |
| ♟️ **에너지 변형 체스** | 2인 | 표준 체스 + 에너지바. 기물 이동마다 점수만큼 에너지 소모, 특수 필드에서 충전·거래. |
| 🕵️ **에너지 변형 마피아** | 8–12인 | 재생에너지 vs 화석연료 vs 산업 스파이. 가이아 HP를 지키며 정체를 추리하는 밤낮 토론 게임. |

- **홈 화면**에서 버튼으로 세 게임 중 하나로 이동
- 각 게임에서 **방 생성** / **방 코드 입력 후 참여**
- **Supabase Realtime** 으로 실시간 동기화, **Vercel** 정적 호스팅으로 무료 배포
- 실물 도구가 전혀 필요 없는 **순수 웹** 구현

---

## 🏗️ 아키텍처

- **순수 정적 사이트(클라이언트 전용)** — 빌드 단계가 없고, Vercel 무료 플랜에 그대로 배포됩니다.
- **호스트 권위(host-authoritative) 모델** — 방을 만든 사람이 게임 상태를 계산하고 Realtime Broadcast 로 전파합니다. 참여자는 액션만 보냅니다.
- **DB 테이블이 필요 없습니다.** Supabase Realtime **Broadcast 채널**만 사용하므로, Supabase 프로젝트의 URL/anon key 만 있으면 동작합니다.
- 마피아처럼 **숨겨진 역할**이 있는 게임은, 호스트만 전체 상태를 보유하고 다른 플레이어에겐 **공개 상태만 전송**하며 개인 역할/결과는 1:1 메시지로 전달합니다.

```
public/
├── index.html              # 홈(게임 선택)
├── css/style.css           # 공용 스타일
├── js/
│   ├── supabase-config.js  # ← Supabase URL/anon key 입력
│   ├── net.js              # 방 생성/참여/실시간 동기화
│   └── ui.js               # 공용 UI 헬퍼(로비/토스트/로그)
└── games/
    ├── gaia/  (engine.js + gaia.js + index.html)
    ├── chess/ (engine.js + chess.js + index.html)
    └── mafia/ (engine.js + mafia.js + index.html)
```

각 게임은 순수 로직 `engine.js` 와 UI 컨트롤러로 분리되어 있습니다.

## 🚀 배포 (Vercel + Supabase, 둘 다 무료)

### 1) Supabase 프로젝트 만들기
1. <https://supabase.com> 에서 무료 프로젝트 생성
2. **Project Settings → API** 에서 두 값 복사
   - `Project URL`
   - `anon public` key
3. Realtime 은 기본 활성화되어 있고 **별도 테이블 설정이 필요 없습니다.**

### 2) 키 입력
`public/js/supabase-config.js` 를 열고 두 값을 붙여넣습니다.

```js
export const SUPABASE_URL = "https://xxxxxxxx.supabase.co";
export const SUPABASE_ANON_KEY = "eyJhbGciOi...";
```

> anon key 는 원래 클라이언트에 노출되도록 설계된 공개 키이므로 정적 사이트에 넣어도 안전합니다.

### 3) Vercel 배포
1. 이 저장소를 GitHub 에 올리고 <https://vercel.com> 에서 **Import**
2. 프레임워크 프리셋: **Other** (빌드 명령/출력 디렉터리 설정 불필요 — `vercel.json` 이 정적 서빙을 처리)
3. Deploy 누르면 끝. 발급된 URL 을 친구에게 공유하세요.

또는 CLI:
```bash
npm i -g vercel
vercel        # 프리뷰
vercel --prod # 프로덕션
```

## 🧪 로컬 실행

```bash
npm run dev    # http://localhost:3000 (npx serve)
# 또는
python3 -m http.server 3000 --directory public
```

- `supabase-config.js` 가 **비어 있으면** 같은 브라우저 안에서만 통신하는 **로컬 테스트 모드**(BroadcastChannel)로 동작합니다. 탭 두 개를 띄워 방 만들기/참여를 시험할 수 있습니다.
- 다른 기기/사람과 함께 플레이하려면 Supabase 키를 채워야 합니다.

## 🎮 게임 규칙 요약

### 가이아 (2–4인)
- 시작 시 **선착순으로 진영(북/남/동/서)** 선택 → 방장이 시작.
- 한 턴에 기물 1개를 **8방향 중 1칸** 이동. 도착 칸의 **자원(●)** 을 수집(+1).
- **화석연료(🛢️) > 재생에너지(🌱)**: 화석연료가 적 재생에너지를 잡으면 **+5자원**. 외교관(🤝)은 싸우지 않고 인접 시 **거래/동맹**.
- **자원 2개**로 화석연료 생성(진영당 최대 40). 화석연료가 많이 모이면 **초대형 인프라(⚡, 3×3)** 로 진화(진영당 최대 3, 전체 5개 도달 시 무작위 3개 붕괴).
- **돌발 이동 / 기후 변수(재생→화석 강제 전환)** 등 랜덤 이벤트. 서부는 모든 이벤트 확률이 높음.
- **40분** 후 자원을 가장 많이 모은 진영 승리.

### 에너지 변형 체스 (2인)
- 방을 만든 사람이 **백(선공)**. 표준 체스 이동/캐슬링/앙파상/승급 그대로.
- 시작 에너지 **10**. 기물 이동 시 점수만큼 소모(퀸=초대형 인프라 9 … 폰=전기 1).
- **⚡ 에너지 필드**: 도달 시 충전. **💱 거래소**: 도달 시 상대와 에너지/기물 협상.
- 에너지가 부족하면 **휴식(+3)** 으로 턴을 넘길 수 있음. **상대 가이아(킹)** 를 잡으면 승리.

### 에너지 변형 마피아 (8–12인)
- 8명 이상 모이면 방장이 시작 → 역할 비밀 배정.
- **밤**: 화석연료(공격)·태양광 패널(보호)·원자력(감청)·산업 스파이(에너지 탈취)가 능력 사용(2에너지). ESG 회사는 매 밤 가이아 HP +0.5.
- **아침**: 주사위로 에너지 생산, 재생에너지 진영은 세금 2에너지 납부(부족 시 가이아 HP −0.5).
- **낮**: 공개 채팅 토론 후 투표로 1명 추방.
- 승리 — 재생에너지: 가이아 생존 + 모든 화석연료 제거 / 화석연료: 가이아 사망 또는 모든 재생에너지 제거 / **산업 스파이: 가이아 사망 시점 에너지 15 이상 보유 시 단독 승리**.

> 비밀 정보(역할·귓속말·감청 결과)는 우측 **🔒 나의 비밀 정보** 패널에만 표시됩니다.

## ⚠️ 참고
- 호스트(방장)가 나가면 해당 방의 게임 상태가 사라집니다(게임당 한 방, 캐주얼 플레이 기준).
- 마피아의 역할 비밀은 일반 UI 사용 기준으로 보호됩니다. 완벽한 부정 방지(개발자 도구 스니핑 차단)가 필요하면 별도 서버 권위 구현이 필요합니다.
